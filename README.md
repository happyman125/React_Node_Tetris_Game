## Overview

Full Stack TypeScript Multiplayer Tetris Game build with Node, socket.io, React and Redux.

Play it at: https://cepalle.github.io/red-tetris/

## Run it locally

````cmd
 npm install
 npm run srv-dist
````

You can now play in http://localhost:8080:4433
